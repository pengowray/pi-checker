About “Missed”

If you skip a digit while typing — say you type the 7th digit but it actually matches the 8th — the checker treats the 7th as missed, and the digit you typed is scored against the 8th instead.

The skipped digit appears as a small inline “ghost” before the digit you actually typed. Missed digits count toward the Missed stat and do not count as wrong.

To trigger missed-detection the checker needs two more matching digits after the skip, so it only kicks in once you have typed enough context.
